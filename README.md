# HR Analytics Dashboard | Python & Tableau

An end-to-end HR analytics project demonstrating the complete data analytics workflow—from synthetic data generation to data cleaning, transformation, and interactive dashboard development. The project leverages Python for data generation and preprocessing, and Tableau for advanced analytics and dashboard visualization.

---

# Project Overview

Organizations rely on HR analytics to monitor workforce trends, improve employee retention, optimize recruitment, and support strategic decision-making. However, obtaining real-world HR datasets can be challenging due to privacy and confidentiality concerns.

This project addresses that challenge by generating a realistic synthetic HR dataset using Python, performing data cleaning and transformation in Jupyter Notebook, and building an interactive HR dashboard in Tableau to provide actionable workforce insights.

This project is intended for:

- HR Professionals
- Data Analysts
- Business Intelligence Analysts
- Tableau Developers
- Recruiters evaluating analytics portfolios
- Students learning HR analytics

# Business Insights

The dashboard enables HR teams and decision-makers to:

- Monitor employee attrition across departments and locations.
- Identify departments with the highest turnover rates.
- Analyze workforce demographics, including gender and age distribution.
- Evaluate salary distribution across job roles and departments.
- Track employee performance and promotion readiness.
- Monitor attendance, overtime, and leave utilization.
- Assess recruitment effectiveness by source and hiring costs.
- Support strategic workforce planning through interactive KPIs and trend analysis.

# How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/HR-Analytics-Python-Tableau.git
```

### 2. Open the Jupyter Notebooks

Run the notebooks to:

- Generate the synthetic HR dataset.
- Clean and transform the data.

### 3. Open Tableau

Open:

```
Tableau/HR_Analytics_Dashboard.twbx
```

### 4. Refresh the data source

If required, reconnect Tableau to:

```
Data/HR_Dataset.xlsx
```

or

```
Data/HR_Dataset.csv
```

### 5. Explore the dashboard

Use the interactive filters to analyze:

- Departments
- Locations
- Gender
- Performance
- Attrition
- Recruitment
- Compensation
